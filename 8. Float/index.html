Float

float merupakan properti CSS untuk mengatur posisi sebuah elemen. Sebuah elemen dapat
dipaksa untuk berada di sebelah kiri dan kanan dari parent/pembungkusnya dengan menggunakan float

nilai float hanya right atau left

perhatikan kotak-kotak berikut. Saya memasang floath right pada kotak merah

yang bisa dilakukan dengan floath:

  • text wrapping
    membuat text mengelilingi gambar / elemen lain
  • image gallery
    membuat serangkaian gambar menjadi galeri
  • multi-column layout
    membuat halaman memiliki beberapa kolom
  • dll
    dan masih banyak lagi
